# Chetris

Tetris project for Advanced Programming lecture

# Keys

### Player 1
|Action|Key|
|---|---|
|Counterclockwise Rotation|Q|
|Clockwise Rotation|W|
|Hard Drop|F|
|Left|C|
|Right|B|
|Soft Drop|V|

### Player 2
|Action|Key|
|---|---|
|Counterclockwise Rotation|L|
|Clockwise Rotation|; (semicolon)|
|Hard Drop|RSHIFT|
|Left|Left Arrow|
|Right|Right Arrow|
|Soft Drop|Down Arrow|
